# 正则表达式的相关概念
## 定义
一种用于匹配、查找、替换的规则
## 常见的正则符号

|正则|含义|
|:---|:---|
|.|任意单个字符|
|\\d|一个数字|
|\\w|一个字母、数字或下划线|
|\\s|一个空白字符|
|[abc]|abc中的任意一个|
|[^abc]|除abc外任意一个|
|[a-z]|小写字母|
|*|前面的内容出现0次或多次|
|+|前面的内容出现1次或多次|
|?|前面的内容出现0或1次|
|{n}|恰好出现n次|
|{n,m}|出现n到m次|
|^|字符串开头|
|$|字符串结尾|

## Java
Java里有两个常用的类：

Pattern

- 表示正则规则

Matcher

- 表示拿这个规则去匹配某个字符串

常见写法：

``` Java
Pattern p = Pattern.compile("正则表达式");
Matcher m = p.matcher("要匹配的字符串");
```

常见用法：

m.find()：查找下一个匹配部分

m.group()：取出匹配到的内容

m.matches()：整个字符串是否完全匹配

String.matches()：直接判断整个字符串是否匹配

### 注意

很多正则本来只需一个反斜杠就行，但在字符串中反斜杠也需转义，因此写成两个反斜杠
